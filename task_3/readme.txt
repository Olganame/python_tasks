13 - 19 вар. Напишите программу, которая считывает текст из файла (в файле может быть больше одной строки) и выводит самое частое слово 
в этом тексте и через пробел то, сколько раз оно встретилось. 
Если таких слов несколько, вывести лексикографически первое (можно использовать оператор < для строк). 
Слова, написанные в разных регистрах, считаются одинаковыми.

Sample Input:
abc a bCd bC AbC BC BCD bcd ABC


Sample Output:
abc 3
